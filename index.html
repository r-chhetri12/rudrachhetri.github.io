Telecom Customer Churn Analysis

SQL Python Power BI(Power Query, DAX)

Analyzed telecom customer data to understand why customers leave and which segments are most at risk. The project focuses on identifying patterns and insights that help improve customer retention.

Tools Used: SQL ( aggregations, CASE statements, Data Cleaning), Python (Pandas, Numpy, Matplotlib, chi square test, analysis), Power BI (Power Query, DAX, interactive dashboards, KPI cards, slicers, charts)

Process: Collected and cleaned ~7,000 customer records. Categorized tenure buckets and contract types, analyzed internet and service usage. Aggregated data using SQL and Python, then built interactive dashboards in Power BI for dynamic exploration.

Insights:

  • Overall churn rate: 27%, meaning 1 in 4 customers leave.
  • 0–1 month tenure customers have the highest churn (62%), indicating onboarding issues.
  • Senior citizens (60+) churned more than younger customers (35% vs 22%).
  • Month-to-month Fiber Optic contracts are the riskiest segments.
  • These insights help design strategies for retention, onboarding, and personalized offers.

HR / Employee Attrition Analysis

SQL Power BI(Power Query, DAX)

Explored employee data to understand attrition patterns, identify high-risk groups, and highlight departments and roles with the most exits.

Tools Used: SQL (data aggregation, filtering, analysis), Power BI (interactive dashboards, charts, slicers, Power Query(Data Cleaning), DAX)

Process: Collected employee datasets, created age, tenure, and department buckets. Analyzed attrition trends using SQL and Python. Built interactive dashboards in Power BI to track attrition by department, role, age, tenure, and gender.

Insights:

  • Overall attrition rate: ~50%+, meaning over half of employees left the company.
  • Highest attrition in the first year, gradually dropping by year 5, indicating onboarding and early engagement issues.
  • Production department and roles like Laborer, Technician, Engineer, and Foreman had the highest exits.
  • Older employees (46+) and high performers showed higher attrition, signaling potential career progression problems.
  • Insights can help HR design better retention strategies, improve onboarding, and reduce turnover.

Call Center Analytics Dashboard

Excel Power Query Pivot Tables & Charts

Created an interactive dashboard to analyze call center performance, including call volume, revenue, call duration, customer satisfaction, and representative performance. The dashboard helps make data-driven operational and strategic decisions.

Tools Used: Excel (Data organization, Pivot Tables, Charts), Power Query (Data cleaning & transformation), Measures / Calculations (Revenue, Average Ratings, Happy Callers), Slicers (Interactive filtering)

Process: Imported and cleaned raw call center data using Power Query. Created Pivot Tables and Charts to summarize call trends, call distribution, revenue by representative, and demographics. Used measures to calculate key metrics and slicers to allow dynamic filtering by representative, city, and time period.

Insights:

  • March and April show the highest call volumes, indicating peak demand periods.
  • Columbus has the highest male caller count, while Cleveland has more female callers.
  • R02 and R03 are the top performers in revenue generation.
  • Average customer rating is 3.9, indicating room for improvement in service quality.
  • Weekends (Saturday & Sunday) have higher call volumes, informing staffing decisions.
